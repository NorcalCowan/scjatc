Ping Localhost

Textbook Reference

Chapter 14 - Connectivity Troubleshooting with Ping (page 270)

Answer

ping localhost

Student Actions

After accessing the command prompt, student should enter the command "ping localhost" which will ping the laptop they are using. This will provide them with the "name" of their laptop.

Textbook Reference

Chapter 14 - Connectivity Troubleshooting with Ping (page 269)

Answer

mobilelab[-#] (should be the same as the label on their laptop)

Student Actions

Locate the information by referencing their ping result.
Instructor Note: Since each person has a different number laptop, the only entry required for a correct result is "mobilelab"

Ping - IPv4 Address

Textbook Reference

Chapter 14 - Connectivity Troubleshooting with Ping (page 270, 271)

Student Actions

Student should reference the textbook on page 270 to locate the switch command "-4" that forces the IPv4 address instead of IPv6.

Textbook Reference

Chapter 14 - Connectivity Troubleshooting with Ping (page 270, 271)

Answer

ping mobilelab-16 -4

Student Actions

Enter answer and click "Submit Answer".
Instructor Note: Answer is not case sensitive, but there SHOULD NOT be a space between "mobilelab" and the hypenated number, but there SHOULD be a space before the "-4" switch.
